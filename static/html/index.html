Frequently Asked Questions

What is BattleSnake?

BattleSnake is a programming competition like no other. Write your snake AI and battle to the death against other teams. Only one snake will emerge victorious.

Do I need to code?

No, but it helps. Teams can include first time prgorammers, designers or anyone who likes playing games. But there will be some programming required.

Preparing for Battle

Participants are strongly encouraged to bring a laptop capable of deploying to Heroku. For more information on getting started, see this guide.

How can I register?

Teams and individual participants can register here.

What are the game rules? How does it work?

Don't worry about it. You'll know when the time comes. To keep things fair, we will go over game rules on stage at the launch of the event. Don't be late.

What technologies will we use?

All snake AIs will deploy using Heroku and play the game using a REST API. You may use the programming language of your choice. We will provide additional docs for Python, Java, Node, and Go.

Can I go alone?

Yes, if you dare.
